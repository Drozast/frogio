# FROGIO Mobile App

App móvil Flutter para FROGIO (iOS, Android, Web).

## Setup

```bash
cd apps/mobile
flutter pub get
flutter run
```

## Build

```bash
# Android
flutter build apk --release

# iOS
flutter build ios --release

# Web
flutter build web
```

## Características

- Clean Architecture
- Bloc para state management
- Multi-plataforma (iOS, Android, Web)
